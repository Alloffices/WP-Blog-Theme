2Installation/Activate Theme

Extract the zipped package downloaded from ThemeForest to your desktop. In the extracted package you will find orchid_wp.zip which is the WordPress theme file.

You can install the theme one of two ways:

1.1 Via FTP: Extract the orchid_wp.zip file and upload the extracted folder to the /wp-content/themes/ folder on your server.

1.2 Via WordPress: Navigate to Appearance > Add New Themes > Upload page. Select the orchid_wp.zip file. Press the Install Now button to upload and install the theme.

2. After uploading the theme, you must activate it. Navigate to the Appearance > Themes page to activate the theme.

3Installation Plugins

After activate theme.

1. Navigate to Appearance > Install Plugins.

2. Press the Install Now button to install the plugins.

3. After plugins installations click on the "Return to Required Plugins Installer"

4. Now you should activate the plugins

  • Content: to import Posts, Pages, Menus
  • Option: to import Theme Options, e.g: site background, typography...

5Building Site

5.1Building Homepage

You can easy to build your Homepage layout by navigate to Appearance > Customize > Site Layout Or You also can do this via Theme Options panel -> Site Layout

If you would like to change the number of posts that should show per page, you can go to Settings > Reading > Blog pages show at most then change the number.

5.2Site Layout

Navigate to Appearance > Customize > Site Layout Or Go to Dashboard > Theme Options > Site Layout

After setting up the main layout of Site. You should save the setting by click Save & Publish button

5.3Header Setting

Navigate to Appearance > Customize > Header Settings Or Go to Dashboard > Theme Options > Header Settings

Setting Header of Site as below:

  • Site logo, if you don't upload the logo image, then your site will show the site name
  • Here, you can easy to edit the top and bottom margin of the Logo

5.4Top Bar Setting

Navigate to Appearance > Customize > Top Bar Settings Or Go to Dashboard > Theme Options > Top Bar Settings

Setting Top Bar of Site as below:

  • Top Bar Social Icons You can Enable / Disable Social Icons that should show on the right side of the Top Bar
  • Search Buttons You can Enable / Disable the search button that should show on the right side of the Top Bar

5.5Featured Slider Setting

Navigate to Appearance > Customize > Featured Slider Settings Or Go to Dashboard > Theme Options > Featured Slider Settings

Setting Featured Slider of Site as below:

  • Turn ON/OFF The Slider on your homepage
  • Select category/categories for the featured slider
  • Select Featured Post IDs - Separate each ID with a comma. These IDs will override the featured category option above.
  • Setup Number of Posts that will be shown in the Slider
  • Display/Remove the Category on the Posts of the Slider

5.6Promo Boxes Setting

Navigate to Appearance > Customize > Promo Boxes Settings Or Go to Dashboard > Theme Options > Promo Boxes Settings

Setting Promo Boxes of Site as below:

  • Turn ON/OFF The Promo Boxes on your homepage
  • There are 3 fields that you should setup for each Promo Box. That is the Promo Box Title, The destination link of Promo Box and the Image Upload field

5.7Post Setting

Navigate to Appearance > Customize > Post Settings Or Go to Dashboard > Theme Options > Post Settings

Setting the Post Display on your Site as below:

  • Turn ON/OFF Featured Image of Post
  • Turn ON/OFF Post Category
  • Turn ON/OFF Post Date
  • Turn ON/OFF Post Comment Link
  • Turn ON/OFF Post Author Name
  • Turn ON/OFF Post Tags
  • Turn ON/OFF Post Share Buttons
  • Turn ON/OFF Post Author Box
  • Turn ON/OFF Post Related Posts Box
  • Turn ON/OFF Post Pagination

5.8Page Setting

Navigate to Appearance > Customize > Page Settings Or Go to Dashboard > Theme Options > Page Settings

Setting the Page Display on your Site as below:

  • Turn ON/OFF Page Share Buttons

5.9Social Media Setting

Navigate to Appearance > Customize > Social Media Settings Or Go to Dashboard > Theme Options > Social Media Settings

Please setup the destination link for each social media that you want to have

5.10Footer Setting

Navigate to Appearance > Customize > Footer Settings Or Go to Dashboard > Theme Options > Footer Settings

5.11Menus

To create Header Menu on the top of Site, you can follow as below steps:

7Language Translation

You can translate WordPress Themes by using PoEdit

So you are looking to change the language of your WordPress based site to something else than English. Let me give you some simple steps to achieve this.

  • The first thing you need to do is to make sure that the theme has the appropriate language file. The default language files and eventually, the bonus ones are located in the theme's directory in the /languages/ directory. They are .po and .mo files.
  • If you don't find the appropriate language file for your target language then you must copy the default.po file and rename the copy accordingly to your desired language. The language files names must have this format "ll_CC", where "ll" is an ISO 639 two-letter language code (lower case) and "CC" is an ISO 3166 two-letter country code (upper case). The country code specification is not redundant: Some languages have dialects in different countries. For example, ‘de_AT’ is used for Austria and ‘pt_BR’ for Brazil. The country code serves to distinguish the dialects. See Language Codes and Country Codes for the lists of codes.
  • So let's say you want to translate to Brazilian Portuguese and you have copied default.po to pt_BR.po. Now you have to edit it using the world's favorite free .po editor, PoEdit (http://www.poedit.net/download.php).

  • Once you open the file you will be presented with the list of all the English strings on the left side and their translation (if present on the right). To translate simply click on a line and on the bottom of the window, you can enter your translation. More info here: http://en.wikipedia.org/wiki/Poedit
  • Once you are done simply hit save in PoEdit and this will automatically save the .po file and generate/regenerate the .mo file (a binary file).
  • Now all you have to do is change the language of your WordPress installation. To do this you need to go to Settings -> General -> Site Language and choose your appropriate language.

You can reopen the .po file at any time and change, continue to translate, etc.
